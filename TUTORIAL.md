# Tutorial

## 1. Build molecular 3D structure

**Apps used: ChemDraw -> Avogadro**

1. Use ChemDraw to draw the 2D structure
2. Save it as .cdxml (for later changes) and export it as .mol (for Avogadro)
3. Import the .mol file into Avogadro
4. Generate 3D coordinates if not yet
5. Pre-optimise the geometry by `Extentions - Optimize geometry`
6. Save it as .cml (for later changes) and export it as .xyz (for ORCA)
